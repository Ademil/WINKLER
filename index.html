📖 Fundamentos Teóricos

O Coeficiente de Winkler (K) — também chamado de módulo de reação do solo — modela o solo como um conjunto de molas independentes, sendo a constante de rigidez de cada mola expressa por unidade de área de contato.

K = q / δ   [kN/m³]
q = pressão de contato [kN/m²]
δ = recalque [m]

CORREÇÃO PARA AREIAS — Terzaghi (1955)

Kar = Kp × [(b + 30) / (2b)]²
b = dimensão menor da fundação [cm]
Kp = K da placa 30×30 cm [kN/m³]

CORREÇÃO PARA ARGILAS — Terzaghi (1955)

Karg = Kp × [(n + 0,5) × 30] / [1,5 × n × b]
n = L / b (relação comprimento/largura)
b = dimensão menor [cm]

SOLO MISTO (ARENOARGILOSO)

Ka = (Kar + Karg) / 2
⚠ Limitações do Modelo de Winkler: O modelo ignora a continuidade do solo e a interação entre molas adjacentes. Em solos com alta rigidez lateral (argila NC, solo cimentado), pode superestimar K. Para estruturas sensíveis, recomenda-se análise com modelo elasto-plástico ou MEF 3D.
